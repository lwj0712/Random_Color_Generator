# Random Color Generator
* 랜덤한 색상을 생성해주는 프로그램 입니다.
* 생성된 색상을 보관, 제거, 초기화 하는 기능이 있습니다.
* 생성된 색상의 헥스코드를 제공합니다.
* 반응형 버튼을 사용했습니다.
